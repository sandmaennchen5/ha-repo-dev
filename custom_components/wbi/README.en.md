# DWD Wildfire & Grassland Fire Index (HACS Integration)

This integration provides:

- `sensor.waldbrandindex`
- `sensor.grasslandfireindex`

Features:

✔ Station List (Dropdown)  
✔ Auto‑GPS‑Detection  
✔ Auto Card Generator  
✔ 4 Lovelace Blueprints  
✔ 4 Lovelace Templates  
✔ HACS-compatible  
✔ Logo  
✔ Release workflow  

## Installation

1. Add repo in HACS  
2. Install integration  
3. Select station or car GPS  
4. Use cards (blueprints orauto-generator)
