# DWD Indice des incendies de forêt et des feux de prairies (intégration HACS)

Cette intégration fournit :

- `indice d'incendie de forêt sensor.`
- `indice de feu sensor.grasland`

Caractéristiques :

✔ Liste des stations (liste déroulante)  
Détection ✔ GPSautomatique  
✔ Générateur decarte automatique  
✔ 4 Blueprints Lovelace  
✔ 4 modèles Lovelace  
Compatible ✔ HACS  
✔ Logo  
Flux ✔ detravail de publication  

## Installation

1. Ajouter un repo dans HACS  
2. Installer l'intégration  
3. Sélectionner la station ou le GPS automatique  
4. Utiliser des cartes (Blueprints ou générateur automatique)
