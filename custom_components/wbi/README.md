# DWD Waldbrand- & Graslandfeuerindex (HACS Integration)

Diese Integration liefert:

- `sensor.waldbrandindex`
- `sensor.graslandfeuerindex`

Features:

✔ Stationsliste (Dropdown)  
✔ Auto‑GPS‑Erkennung  
✔ Auto‑Card‑Generator  
✔ 4 Lovelace‑Blueprints  
✔ 4 Lovelace‑Templates  
✔ HACS‑kompatibel  
✔ Logo  
✔ Release‑Workflow  

## Installation

1. Repo in HACS hinzufügen  
2. Integration installieren  
3. Station auswählen oder Auto‑GPS  
4. Karten nutzen (Blueprints oder Auto‑Generator)
